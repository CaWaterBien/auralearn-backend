Task: Create, Save, and Preview an HTML file

Follow these grader-checked steps. This page helps you complete them instantly.

  1. Create a new HTML file. (Use the example below or your own code.)
  2. Add the required tags: <!DOCTYPE html>, <html>, <head>, and <body>.
  3. Add a title: inside <head> put <title>My First Webpage</title>.
  4. Display a heading: inside <body> put <h1>Hello, World!</h1>.
  5. Save your file as index.html and open it in a browser to preview.

Example HTML (index.html)

<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <title>My First Webpage</title>
</head>
<body>
    <h1>Hello, World!</h1>
</body>
</html>
